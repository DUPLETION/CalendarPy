# Python Learn

Android-приложение для изучения Python с ежедневными уроками и мотивационными напоминаниями.

## Возможности

- **8-недельный курс** обучения Python
- **Ежедневные уведомления** с мотивационными сообщениями
- **Отслеживание прогресса** - отметка выполненных дней
- **Карточки дней** с теорией, практикой и задачами
- **Современный дизайн** в стиле Material Design

## Структура курса

| Неделя | Тема |
|--------|------|
| 1 | База Python |
| 2 | Структуры данных и функции |
| 3 | ООП |
| 4 | Практика и алгоритмы | Библиоте |
| 5ки |
| 6 | Telegram-бот |
| 7 | База данных |
| 8 | Финальный проект |

## Технологии

- **Kotlin** - основной язык разработки
- **Android SDK** - для нативного Android
- **Chaquopy** - Python для Android
- **Material Components** - дизайн

## Установка

1. Клонируйте репозиторий
2. Откройте проект в Android Studio
3. Соберите и запустите на устройстве

## Настройка уведомлений

- В настройках приложения можно включить/выключить уведомления
- Установить время напоминания
- Отправить тестовое уведомление
